# CareerCoach.ai

AI that fixes your CV, interview answers, and LinkedIn — instantly.

## Features
- Instant ATS score based on CV and job post
- Bullet rewrite suggestions
- Interview question generator
- Next.js 14 + TypeScript + Serverless API

## Run locally
```
npm install
npm run dev
# visit http://localhost:3000
```

## Deploy
1. Push to GitHub
2. On Vercel → “Import Project” → select this repo
3. Add `OPENAI_API_KEY` (optional)
4. Deploy
